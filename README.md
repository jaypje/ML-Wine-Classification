# Wine Classification with SVM

Classifying 3 types of wine using Support Vector Machine (SVM)

## Dataset
UCI Wine dataset from sklearn (178 samples, 13 features)

## Steps
- Exploratory data analysis (heatmap, bar charts)
- MinMaxScaler normalisation
- SVM classification
- 91-100% accuracy across different train/test splits

## Libraries
scikit-learn, pandas, numpy, matplotlib, seaborn
